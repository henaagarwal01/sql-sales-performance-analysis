# 📊 Sales Performance Analysis (SQL Project)

## 📌 Overview  
Analyzed a retail sales dataset using SQL to evaluate revenue, profitability, and discount impact. The project focuses on generating business insights from raw data.

---

## 🛠 Tools Used  
- PostgreSQL (pgAdmin4)  
- SQL  
- Kaggle Superstore Dataset  

---

## 🧠 SQL Skills  
- GROUP BY, ORDER BY  
- Aggregations (SUM, AVG)  
- HAVING  
- CASE WHEN  
- Window Functions (RANK)  
- CTE (WITH)

---

## 🔍 Key Analysis  
- Total sales and profit  
- Top revenue and profitable cities  
- Loss-making cities  
- Discount vs profit analysis  
- Category & sub-category performance  
- Regional analysis using CTE  

---

## 📈 Key Insights  
- High discounts (>30%) reduce profitability  
- Some cities have high sales but overall loss  
- Tables and Bookcases are loss-making  
- West region performs best  

---

## 💡 Conclusion  
This project demonstrates how SQL can be used to solve business problems, identify risks, and support decision-making.
